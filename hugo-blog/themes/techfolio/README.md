# Techfolio Hugo Theme

A custom Hugo theme matching the deep blue tech aesthetic of the main portfolio site.

## Features

- Deep blue color scheme (#020c1b, #0a192f)
- Neon cyan accents (#00f3ff, #64ffda)
- Saira + Roboto fonts
- Responsive design
- Bilingual support (zh-TW / English)
- Card-based post layout
- Clean single post design

## Usage

This theme is designed to be used within the main portfolio repository.
